新やあ鯖について

新やあ鯖はサバイバルを軸に運営しています！
Minecraft Java Edition向けにサーバーを運営していますが、プラグインを導入して統合版でも遊べるようにしています！

サーバー規約

埋め込みでの読字が難しい場合こちらのサイトでお読みください。

参加方法

このサーバーではJava版と統合版両方で参加できます。それぞれの参加方法を紹介します。

Java版

推奨サーバーアドレス : game.ssnetwork.io:40100
予備用サーバーアドレス : 25.ip.gl.ply.gg:43340
推奨バージョン : 1.20.1 (ViaVersionが入っているため1.20.1以上ならできると思います)

Java版は以下の手順で参加してください

  1. Minecraftを起動してください
  2. 起動したら「マルチプレイ」を押してください
  3. この画面になったら「サーバーを追加」、または「ダイレクト接続」を押してください(サーバーを追加の前提で解説します)
  4. サーバーアドレスの欄に「game.ssnetwork.io:40100(または25.ip.gl.ply.gg:43340)」と入れて「完了」を押してください(サーバー名は自由です)
  5. これで終わりです。もし接続できない場合はサーバーアドレスを確認の上入りなおしてください。その上で参加できない場合は管理者までお知らせください。

統合版

サーバーアドレス: 18.ip.gl.ply.gg
ポート: 4892
推奨バージョン: 通常版の最新バージョンをお使いください(開発版はサポートしていません)

  1. Minecraftを起動してください
  2. 起動したら「プレイ」を押してください
  3. この画面になったら「サーバー」を押してください
  4. そして「サーバーを追加」を押してください
  5. サーバーアドレスの欄に「18.ip.gl.ply.gg」、ポートの欄に「4892」と入れて「完了」を押してください(サーバー名は自由です)
  6. これで終わりです。もし接続できない場合はサーバーアドレスを確認や、SwitchなどのPC以外の機器では本体設定を確認の上入りなおしてください。その上で参加できない場合は管理者までお知らせください。
